# Web Based Calculator

This web based calculator used for calculation. It have the following functionality:

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Percentage calculation
6. Show results
7. Delete wrongly inputed number
8. Clear output

## Tools and Technologies

1. Vite.js
2. React.js
3. TypeScript
4. TailwindCss
